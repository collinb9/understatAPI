understatapi.utils.unpack_dataframe(data)[source]¶

Unpack each column in a dataframe whose entries are dictionaries into separate columns. The new columns are named by combining the original collumn name with the corresponding key in the dictionary

Parameters

data (DataFrame) – A dataframe with nested columns

Return type

pd.DataFrame

Example

>>> import pandas as pd
>>> data= pd.DataFrame(
...     data=[[{"ham": 1, "eggs": 2}]],
...     columns=["spam"]
... )
>>> print(data)
                    spam
0  {'ham': 1, 'eggs': 2}
>>> unpacked_data = unpack_dataframe(data)
>>> print(unpacked_data)
   spam_ham  spam_eggs
0         1          2
